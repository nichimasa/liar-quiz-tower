# ナンバークラフト

数字と演算子を組み合わせて、目標値を作るスマホ向けクイズゲームです。

## 遊び方

`index.html` をブラウザで開くとすぐに遊べます。

- 難易度は `初級`、`中級`、`上級` の3段階
- 初級30問、中級50問、上級80問
- 4つの数字をすべて1回ずつ使用
- `+`、`-`、`×`、`÷`、`()` を使って `10` を作成
- 問題表示から正解までの時間を記録
- ホーム画面で難易度別の正解数と最速タイムを表示
- ヒント、正解表示、全正解パターン表示に対応

## ファイル構成

- `index.html`: 画面構造
- `style.css`: スマホ向けレイアウトと見た目
- `script.js`: 問題データ、式作成、判定処理
