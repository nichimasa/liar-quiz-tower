# ナンバークラフト

数字と演算子を組み合わせて、目標値を作るスマホ向けクイズゲームです。

## 遊び方

`index.html` をブラウザで開くとすぐに遊べます。

- 難易度は `初級`、`中級`、`上級` の3段階
- 4つの数字をすべて1回ずつ使用
- `+`、`-`、`×`、`÷`、`()` を使って目標値を作成
- 回答開始から正解までの時間と連続正解数を記録
- ヒント、正解表示、全正解パターン表示に対応

## ファイル構成

- `index.html`: 画面構造
- `style.css`: スマホ向けレイアウトと見た目
- `script.js`: 問題データ、式作成、判定処理
